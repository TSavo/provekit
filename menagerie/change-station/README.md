# Change Station

Change Station is the planned Menagerie destination for proof-carrying commits.

Its core claim is that a change is a witnessed `p -> q` transition: the old
artifact, new artifact, and preserved or repaired obligations are all named by
content-addressed evidence.

Expected exhibits:

- commit-level implication receipts;
- patch review that checks proof deltas;
- regression examples where the transition fails to carry the required claim.
