# Supply Chain Rails

Supply Chain Rails is the planned Menagerie destination for proof-carrying
artifact admission.

Its core claim is that packages, binaries, build outputs, CI inputs, and
witnesses stay on verified tracks when rank-3 pins, `binaryCid`, and CICP agree
on the same byte-identified closure.

Expected exhibits:

- source-to-binary admission receipts;
- CI reuse allowed only for byte-identical inputs;
- attack sketches where a substituted artifact leaves the rails.
