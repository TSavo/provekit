# Bridgeworks

Bridgeworks is the planned Menagerie destination for proof preservation across
domain crossings.

Its core claim is that a proof can move between language runtimes, package
ecosystems, protocol formats, repository boundaries, and CI systems without
becoming a hand-waved translation.

Expected exhibits:

- cross-language LinkBundle receipts;
- package and protocol boundary bridges;
- failure receipts for bridges that drop or weaken obligations.
