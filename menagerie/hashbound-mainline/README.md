# Hashbound Mainline

Hashbound Mainline is the planned Menagerie destination for long proof routes
that cross domains while retaining a compact verification handle.

Its core claim is that an implication chain can pass through software,
compiler, architecture, hardware, and physics evidence while the consuming
software only needs the 64-byte content-addressed handle for the carried proof.

Expected exhibits:

- toy vertical stack from source contract to compiled artifact to execution
  assumption;
- real-world stack map for compiler, CPU, and paper witnesses;
- receipt checks that distinguish "chain exists" from "claim verified".
