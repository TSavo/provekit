# Java Real-Library Logo: Gson

This showcase proves the Java seat on Gson 2.10.1 with zero source changes from
the official upstream `gson-parent-2.10.1` tag. `run.sh` fetches the pinned tag
zip plus the real JUnit jar, copies the real Gson core sources and the real
`JsonStreamParserTest.java`, and verifies the durable `.proof` plus witness
bundle.

The consistency axis learns JUnit4 assertion vocabulary with `javap` from the
real JUnit jar and proves exact assertion rows from `JsonStreamParserTest.java`.
The witness axis compiles the real Gson core source plus that real JUnit class.
Truth-based Gson tests are outside this receipt's assertion vocabulary and are
not claimed here.
